194 /*
195  Consider the following code:
196  static inline void foo() { bar(); }
197  when foo() is never called.
198 
199  With gcc, foo() is a local static function, so the dependencies
200  are optimized away at compile time, and there is no dependency on bar().
201  With other compilers (HP, Sun Studio), the function foo() implementation
202  is compiled, and bar() needs to be present to link.
203 
204  Due to the existing header dependencies in MySQL code, this header file
205  is sometime used when it is not needed, which in turn cause link failures
206  on some platforms.
207  The proper fix would be to cut these extra dependencies in the calling code.
208  DISABLE_MYSQL_THREAD_H is a work around to limit dependencies.
209  DISABLE_MYSQL_PRLOCK_H is similar, and is used to disable specifically
210  the prlock wrappers.
211 */
